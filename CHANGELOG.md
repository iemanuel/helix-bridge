# Changelog

## 1.0.0 (2026-07-02)

Initial release. Helix Bridge is a solar inverter monitoring bridge running as a Home Assistant add-on.

- **17 simulation profiles** covering all supported inverter types
- **13 collector drivers** for Sunsynk/Deye/Sol-Ark, Goodwe, Huawei, Solis, Growatt, Sofar, Luxpower, SMA, Fronius, Voltronic/Axpert/MPP Solar, Victron Energy, and generic SunSpec/Modbus inverters
- **MQTT auto-discovery** with Home Assistant
- **Pre-built HA dashboard** in `ha-dashboards/helix_bridge.yaml`
- **REST API** at `/api/v1/metrics`
